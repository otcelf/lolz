# 🔥 Lolz Team Bot

P2P-гарант бот для безопасных сделок в Telegram.

---

## 🚀 Деплой на bothost.ru

1. Залей репозиторий на GitHub
2. На bothost.ru создай нового бота:
   - **Платформа:** Telegram
   - **Язык:** Python 3.11
   - **Git URL:** `https://github.com/ТВО_ЮЗ/ИМЯ_РЕПО.git`
   - **Ветка:** `main`
3. В поле **Bot Token** вставь токен бота
4. В **Переменные окружения** добавь:

| Ключ | Значение |
|------|----------|
| `BOT_TOKEN` | токен от @BotFather |
| `ADMIN_IDS` | `8659836741` |
| `DB_PATH` | `database.db` |
| `BANNER_FILE_ID` | file_id баннера (необязательно) |

5. Нажми **Создать** — бот запустится автоматически

---

## 💻 Локальный запуск

```bash
pip install -r requirements.txt
cp .env.example .env
# Заполни .env своими данными
python bot/main.py
```

## ⚙️ Функционал

- 📝 Создание сделок по типам товаров (NFT, аккаунты, Premium, Stars и др.)
- 🔗 Ссылка на сделку для покупателя

- 💳 Реквизиты с валидацией (карта 16 цифр, TON адрес и др.)
- ⚙️ Воркер-меню для участников команды
- 🛡 Верификация продавцов
- 🔗 Реферальная программа
- 📩 Центр обращений
- 🌐 Мультиязычность
- ⚙️ Панель администратора
